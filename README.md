# Desafios React Front

Projeto desenvolvido para a atividade de desafios de Front-End com React.

## Tecnologias utilizadas

- React
- Vite
- Axios
- React Router DOM
- JSON Server

## Funcionalidades

- Mensagem exibida ao abrir a aplicação usando useEffect
- Listagem de produtos usando axios.get
- Cadastro de produto usando axios.post
- Remoção de produto usando axios.delete
- Navegação entre páginas usando React Router

## Como rodar o projeto

Instale as dependências:

```bash
npm install
```
